# Audio Diarizer

## To setup app

> Extract compressed file in models directory

`pip install virtualenv`

`python -m venv venv`

`./venv/Script/activate`

`pip install click`

`pip install vosk`

> make sure you're in the app dirrectory where the setup script is located.

`pip install --editable .`

## Available commands

* `show-speaker`
>    arguements:
        `-a`
        `--audio`
* `show-word`
>    arguements:
        `-a`
        `--audio`
